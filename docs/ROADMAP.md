# Roadmap

## Motion Extraction

- Add DuoMo backend.
- Add WHAM backend.
- Add HMR/HMR2 backend.
- Add multi-person tracking and identity selection.

## Retargeting

- Add GMR backend.
- Support custom humanoid URDF/MJCF retargeting.
- Add learned or force-aware contact classification and trajectory smoothing beyond the current geometry-and-speed contact report.
- Add `.motion` export for ProtoMotions-style downstream training.

## Simulation

- Add controller-driven Isaac Sim playback.
- Add MuJoCo replay backend.
- Add PyBullet backend.
- Add Genesis backend.
- Add automatic video rendering and screenshot capture.

## Robots

- Support Unitree H1/H1-2.
- Support Booster/T1 style humanoids.
- Support custom URDF/MJCF configuration.

## Reproducibility

- Add Docker/Apptainer recipe where licenses permit.
- Add dataset example manifests.
- Add benchmark reports for extraction and retarget quality.
